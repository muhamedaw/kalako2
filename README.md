# kalako2

Built with Builder OS.

## Setup

```bash
cp .env.example .env
bash scripts/dev.sh
```

## Scripts

| Script | Purpose |
|--------|---------|
| scripts/dev.sh | Start dev environment |
| scripts/build.sh | Build for production |
| scripts/test.sh | Run tests |
| scripts/audit.sh | Project health audit |
| scripts/clean.sh | Remove build artifacts |
